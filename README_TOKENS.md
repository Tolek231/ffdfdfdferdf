
# Токен‑кабины (multi-tenant)

Добавлен режим `tokens` для показа статистики по конкретным токенам.

## Как это работает

- В каталоге `DATA_DIR` заведите файл1 `tokens` (без расширения). В нём — по одному токену на строку:
  ```
  o4ko
  4ort
  ```
- Если файл пустой или отсутствуе1т — разрешены любые токены (режим совместимости).

## Отправка письма с токеном

В запрос `/api/send` добавьте поле `token`:

```json
{ "email": "user@example.com", "text": "Привет!", "token": "o4ko" }
```

Сервер сохранит `record.token`, передаст `app_token` в `custom_args` SendGrid и будет начислять счётчики как в **общую** статистику, так и в разрезе токена.

## Просмотр статистики

- Общая сводка: `https://sender-screens.amvera.io/`
- По токену: `https://sender-screens.amvera.io/o4ko` (любой первый сегмент пути — это токен).

Фронтенд сам добавляет `?token=...` при обращении к `/api/series`.
